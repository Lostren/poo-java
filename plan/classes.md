# Classes

## Personagem (classe base)

### Atributos

- Vida
- Mana
- Força
- Defesa
- Agilidade
- Poder Mágico

### Métodos Comuns

- `atacar()`
- `usarHabilidade()`
- `defender()`
- `correr()`
- `receberDano()`
- `subirNivel()`

## Povos Livres

### Elfo

| Campo | Valor |
|-------|-------|
| **Papel** | Rápido / Mágico / Ofensivo |
| **Atributos Fortes** | Agilidade, Poder Mágico, Mana |
| **Atributos Fracos** | Vida, Defesa |
| **Estilo de Combate** | Combate ágil e preciso. Uso de arcos, lâminas leves e magia élfica. |
| **Afinidade Mágica** | Alta |
| **Mobilidade** | Muito alta |
| **Resistência** | Baixa. Depende mais de esquiva, velocidade e posicionamento. |
| **Habilidade Inicial** | Disparo Rápido |
| **Equipamentos** | Arcos, espadas leves, armaduras leves, artefatos élficos |
| **Complexidade** | Média |
